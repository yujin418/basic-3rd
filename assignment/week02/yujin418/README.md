# 과제

## 목표  
- Wireshark를 이용하여 PDF 파일 다운로드 패킷 캡처  
- HTTP 상태 코드 분석  

### 1. Wireshark 실행 및 캡처 시작  
wifi를 사용 중이므로 해당 인터페이스를 선택하고 캡처를 시작하였다.

### 2. 디스플레이 필터 설정
HTTP 패킷만 보기 위해 필터를 설정하였다.

### 3. 사이트 접속 및 암호화 해제  
용인시 홈페이지 접속
HTTPS를 HTTP로 변경하여 패킷이 보이도록 하였다.

### 4. PDF 파일 다운로드  
다운로드 과정에서 경고창이 발생하였고, 계속 버튼을 눌러 진행하였다.

### 5. 상태 코드 분석  
- 2xx: success (200 OK)  
- 3xx: Redirection (304 Not Modified)  
- 4xx, 5xx: 오류 없음

응답 총 77건 확인.
리디액션 1건 확인. 브라우저 캐시에 이미 동일한 파일이 존재하여 서버가 데이터를 재전송하지 않고 로컬 데이터를 사용하도록 안내한 결과로 보임

또한 HTTP는 포트 80, HTTPS는 포트 443을 사용하는 것을 확인하였다.

### 6.패킷 캡쳐 종료 후 저장
.pcapng로 저장
