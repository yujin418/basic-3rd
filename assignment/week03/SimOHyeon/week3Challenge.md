# 1. 기본 분석

- 사용된 프로토콜을 3가지 이상 작성하시오.

1. TCP
: 전송 제어 프로토콜로 인터넷상에서 데이터의 신뢰성 있는 전송을 보장하는 핵심 통신 규약이다.

2. HTTP
: 클라이언트와 서버가 HTML, 이미지 등 리소스를 주고받기 위해 사용하는 프로토콜이다.

3. DNS
:사람이 기억하기 쉬운 도메인 이름을 컴퓨터가 이해하는 IP 주소로 변환해 주는 프로토콜이다.
